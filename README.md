# 可见的智能：机器学习算法可视化实验室

作者：HaoranWang

在线访问：https://haoranwang-coder.github.io/visible-ai-lab/

项目仓库：https://github.com/HaoranWang-coder/visible-ai-lab

## 项目简介

“可见的智能”是一个纯前端机器学习算法可视化项目。项目通过 HTML、CSS、JavaScript 和 Canvas，把常见机器学习算法的运行过程变成可以观察、可以点击、可以调参数的互动实验，适合中学生、初学者和比赛答辩展示。

## 功能模块

1. 首页
   - 项目简介
   - 作者信息
   - GitHub 仓库链接
   - 六个算法实验入口
   - Canvas 算法示意图
   - 动态指标、可调实验、过程可见三个亮点说明

2. K-Means 聚类实验室
   - 随机生成二维散点
   - 调节 K 值和数据点数量
   - 初始化聚类中心
   - 单步迭代和自动运行
   - 显示点到聚类中心的连线
   - 显示聚类中心移动轨迹
   - 实时显示簇内平方误差和每簇点数

3. DBSCAN 密度聚类实验室
   - 调节 eps 邻域半径
   - 调节 minPts 最少邻居数
   - 单步扫描和自动聚类
   - 显示核心点、边界点和噪声点
   - 显示当前扫描点的 eps 范围和邻居连线

4. KNN 分类实验室
   - 自动生成红色、蓝色两类样本
   - 调节 K 值和每类样本数量
   - 点击画布添加测试点
   - 高亮最近 K 个邻居并绘制连线
   - 显示多数投票分类结果和投票置信度
   - 可开启决策边界背景，观察分类区域变化

5. 朴素贝叶斯分类实验室
   - 生成两类概率样本
   - 调节每类样本数和类别分散度
   - 点击画布计算测试点后验概率
   - 显示红类概率、蓝类概率和最终分类
   - 可开启概率决策区域背景

6. 感知机二分类实验室
   - 生成线性可分的红蓝样本
   - 调节学习率、样本数量和类别间隔
   - 训练一步、训练一轮和自动训练
   - 显示决策边界、错误样本和最近更新样本
   - 实时显示 w1、w2、b、准确率和错误样本数

7. 线性回归实验室
   - 生成带噪声的线性数据
   - 调节学习率和数据噪声
   - 显示拟合直线和预测误差线
   - 使用梯度下降训练模型
   - 支持训练一步、训练 50 步和自动训练
   - 实时显示 w、b、loss、训练轮数和当前公式
   - 使用小型 Canvas 绘制 loss 变化曲线

## 技术栈

- HTML
- CSS
- JavaScript
- Canvas

项目不使用后端、不使用数据库、不使用 Python、不使用 React/Vue、不使用 npm 或构建工具。

## 本地运行方式

方式一：直接打开

1. 下载或克隆项目代码。
2. 在文件夹中双击 `index.html`。
3. 在浏览器中选择不同实验页面。

方式二：使用本地静态服务

如果希望使用本地网址访问，可以在项目目录中启动任意静态服务器，然后打开 `index.html`。项目本身不依赖任何构建步骤。

```powershell
python -m http.server 5500 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:5500/index.html
```

## GitHub Pages 部署方式

1. 将代码上传到 GitHub 仓库。
2. 进入仓库的 Settings。
3. 进入 Pages。
4. Source 选择 Deploy from a branch。
5. Branch 选择 main。
6. Folder 选择 /root。
7. 保存后等待 GitHub 生成访问链接。

仓库地址：

```text
https://github.com/HaoranWang-coder/visible-ai-lab.git
```

## 项目亮点

- 纯前端实现，适合部署到 GitHub Pages。
- 六个经典算法都是真实计算过程，不是静态演示。
- Canvas 实时绘制数据点、聚类中心、中心轨迹、分类邻居、决策边界、概率区域、回归直线和 loss 曲线。
- 参数控制区、可视化区、算法解释区三栏布局清晰。
- 所有文字均为中文，便于课堂讲解和比赛答辩。
- 代码结构简单，变量命名清晰，适合初学者阅读和继续扩展。
